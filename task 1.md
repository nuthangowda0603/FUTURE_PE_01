## 🧠 Prompt Engineering Workflow

### Step 1 — Brand Briefing Prompt
```
You are a UGC ad copywriter for a D2C skincare brand called GlowLab.
Product: Daily SPF Moisturizer with Niacinamide (₹599)
Target audience: Indian women and men aged 18–30 with oily or combination skin.
Tone: Authentic, relatable, first-person UGC style — NOT corporate or salesy.
Goal: Drive clicks and purchases via Instagram Reels and Facebook Ads.
```

### Step 2 — Hook Generation Prompt
```
Generate 6 scroll-stopping hooks for a UGC ad about GlowLab's SPF Moisturizer.
Each hook must:
- Be under 10 words
- Create curiosity or pain-point recognition
- Feel like something a real user would say (not an ad)
- Vary in style: question, bold claim, contrast, confession, stat, story
```

### Step 3 — Full Script Prompt
```
Write a 30-second UGC ad script for GlowLab SPF Moisturizer using:
Hook: [chosen hook]
Format: Problem → Agitation → Solution → CTA
Tone: Casual, first-person, like a friend recommending a product
Platform: Instagram Reels
Include: natural pauses, gestures cues in brackets, and a strong CTA at the end
```

### Step 4 — Platform Adaptation Prompt
```
Adapt the following UGC script for [Facebook / YouTube Shorts].
Adjust: length, tone formality, CTA style, caption, and hashtags.
Keep the core message intact but match the platform's content norms.
```

## 🎨 Brand Voice Guide (Prompt Engineering Reference)

| Brand Type | Tone Descriptor | Prompt Instruction |
|------------|----------------|-------------------|
| D2C (GlowLab) | Relatable, honest, friend-like | "Write as if you're texting a friend about a product you love, not writing an ad" |
| Local Business | Warm, community-first | "Use neighborhood language, mention the local area, keep it personal" |
| SaaS | Smart, efficiency-driven | "Focus on ROI, time saved, and professional outcomes. No fluff." |
| Creator Brand | Hype-driven, exclusive | "Use FOMO, scarcity, and community identity. Bold and energetic." |

## 📊 Ad Performance Framework

| Element | What to Test | Why It Matters |
|---------|-------------|---------------|
| **Hook** | 3–6 variations | Hook determines if user watches or scrolls |
| **Length** | 15s vs 30s vs 60s | Match to platform attention span |
| **CTA** | "Link in bio" vs "Shop now" vs "DM me" | Affects click-through rate |
| **Tone** | Educational vs Confessional vs Transformational | Different audiences respond differently |
